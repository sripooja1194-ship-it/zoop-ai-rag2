# Zoop AI RAG Assistant 🤖

An AI-powered document chatbot built using Streamlit, Groq LLM, and FAISS for Retrieval-Augmented Generation (RAG).

---

## 🚀 Features
- 📄 Chat with PDFs
- 🤖 AI-powered answers using Groq LLM
- 🔍 Semantic search using FAISS
- 🎤 Voice input support (Whisper)
- 🖼 Image upload support
- 💬 Chat history memory
- ⚡ Fast responses

---

## 🛠 Tech Stack
- Python
- Streamlit
- Groq API (LLM)
- FAISS (Vector Database)
- LangChain
- Whisper (Speech-to-text)
- HuggingFace Embeddings
- PIL (Image processing)

---

## 📌 How it works
1. Upload PDF files
2. Text is split into chunks
3. Embeddings are created
4. FAISS stores vectors
5. User asks question
6. Relevant chunks are retrieved
7. LLM generates final answer
